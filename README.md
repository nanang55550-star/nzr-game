# nzr-game
Berisi game sederhana yang dapat di jalankan secara offline cocok untuk ketika gabut tanpa internet 

## instalasi
```bash
git clone https://github.com/nanang55550-star/nzr-game.git
```
### masuk ke folder 
```bash
cd nzr-game
```
kalau main game multiplayer harus install ini
```bash
pip install websockets
```
❗ catatan: kalau mau update Repository github karena ada game baru gunakan perintah ini ❗
```bash
git pull origin main
```
## cara pakai 
apa game yang ingin anda mainkan lihat dengan perintah ls
```bash
ls
```
lalu masuk ke folder yang game yang anda mau ingin kan
```bash
cd [nama folder game]
```
contoh:
```bash
cd tetris
```
jalan skrip dengan:
```bash
python server.py
```
